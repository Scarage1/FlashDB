Quickstart Checklist

Run a clean onboarding flow in less than a minute.

  1. Open the Console and run PING to verify connectivity.
  2. Create a key: SET demo:user "Ada".
  3. Inspect values and TTL in the Keys explorer.
  4. Expire test data with EXPIRE demo:user 60.

Operational Safety

Keep production stable with these guardrails.

  1. Use key patterns (e.g., session:*) before bulk actions.
  2. Confirm destructive actions like FLUSHDB.
  3. Monitor memory usage in the Overview panel.
